Important Product Notes

Honest roadmap matrix and portfolio stack constraints.

  • ✦ This admin dashboard is a standalone preview page. It exists to display UI quality and is not protected by guards.
  • ✦ OAuth sign-in: Apple and Google authentication options remain visual placeholders only.
  • ✦ SMS verification: The verification flow is mocked/planned for future stages.
  • ✦ Billing integration: Pricing lists are preview designs. No production payment gateway is connected.
  • ✦ End-to-End Encryption (E2EE): Message protocols do not execute full crypto-ciphers in this stack.
  • ✦ Database Stack: Local SQL database is configured for development purposes only.
  • ✦ Backend Deploys: The ASP.NET Core API is designed for local setup and is not deployed to production.
  • ✦ Netlify Hosting: The live public portfolio preview remains fully operational using Local Mode.

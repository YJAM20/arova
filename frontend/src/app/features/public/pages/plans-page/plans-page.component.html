Local Mode

Local Mode

A browser-local demo space for exploring Arova instantly.

  • ✔ Runs in this browser.
  • ✔ No backend required.
  • ✔ Good for demos and testing.
  • ✔ Data stays in localStorage.
  • ❌ Does not sync across devices.
  • ❌ Does not support real partner accounts across devices.
  • ❌ No real email/SMS/push delivery.
API Mode

API Mode

A backend-connected mode for real app development and production-style usage (backend required).

  • ✔ Connects to configured backend.
  • ✔ Enables database-backed storage.
  • ✔ Enables real authentication flow if backend configured.
  • ✔ Enables SignalR/realtime features if backend running.
  • ✔ Enables future email/reminder integrations if configured.
  • ⚠ Requires backend setup.
  • ❌ Does not work if backend is offline.

Frequently Asked Questions

Does Local Mode sync across devices?

No. Local Mode keeps data in this browser only.

Does API Mode require a backend?

Yes. API Mode needs the configured Arova backend to be running.

Are payments real in this demo?

No. Real billing is not active in this demo unless a payment provider is explicitly implemented and configured.

Is Arova end-to-end encrypted?

True production E2EE is planned, but not active in this demo.

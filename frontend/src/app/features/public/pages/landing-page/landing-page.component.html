About Arova

A private shared space for two people in a relationship.

Arova helps partners remember, communicate, plan, and feel closer without forcing romance or turning care into performance. It provides a digital sanctuary away from general social feed noise.

Privacy & Transparency

Private by design, honest about what is implemented.

True end-to-end encryption is a roadmap item, not a current claim. Arova supports couple-scoped data, Local Mode storage, API Mode database configurations, and secure WebSocket chat. All communications use standard transport layer security.

Roadmap & Coming Soon

Phase 1 (Active)

Standalone Local Mode, API mode sync, SignalR live chat, letters vault, reasons log, and settings customization.

Phase 2 (Planned)

Cryptographic client-side E2EE implementation, Apple & Google sign-in templates, and SMS pairing confirmation gateways.

Phase 3 (Mobile)

Native iOS & Android mobile wrappers (Capacitor), push notifications, and widget shortcuts for home screens.

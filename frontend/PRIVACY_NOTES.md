# Privacy Notes

Arova is a generic couples app codebase prepared for portfolio and development use.

Do not commit real names, private letters, personal notes, birthdays, relationship dates, emails, phone numbers, real locations, passcodes, secrets, exported backup JSON files, screenshots, photos, audio, or other private media.

This version contains no real payment flow. Google and Apple auth are placeholders unless configured in a real backend environment. Phone verification is also a placeholder; email verification uses backend development codes.

Keep personal versions in a separate private folder or private repository.

LocalStorage data stays on the current browser and device. API Mode data lives in the backend. Neither mode should be assumed to provide true end-to-end encryption unless that is implemented later.
